О проекте

Этот проект представляет собой автономную метеостанцию, созданную для мониторинга погодных условий в режиме реального времени и предоставления прогноза с помощью веб-интерфейса, разработанного на Django.

Цель

Основная цель — обеспечить непрерывный сбор данных о погоде и визуализировать их на сайте и хранить в базе данных

Состав системы

  • Arduino Uno — микроконтроллер для опроса датчиков
  • ESP8266 — модуль передачи данных по Wi-Fi
  • Датчики:
    • BME280 — температура, влажность, давление
    • GY-302 — уровень освещённости (облачность)
  • Django + PostgreSQL — серверная часть и база данных
  • Интерфейс Bootstrap — адаптивный фронтенд

Возможности

  • Просмотр текущей температуры, влажности, давления, облачности и ветра
  • График температур и таблица данных
  • Фильтрация по дате и экспорт данных в CSV

Планы на развитие

В будущем планируется реализовать мобильное приложение, расширить список датчиков, добавить энергонезависимое питание, а также внедрить машинное обучение для прогноза на основе собственных данных.

Исходный код

Проект доступен на GitHub. Любой желающий может ознакомиться с исходным кодом и предложить улучшения.
